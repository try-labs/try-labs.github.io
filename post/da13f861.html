MySQL 删除 relay log 的方法

方法1 ：自动 设置参数 relay_log_purge = 1

这个参数需要重启数据库才可生效。默认情况下relay-log-purge 是开启的。
如果你对从数据库要求不高，可以将relay_log_purge = 1。
如果数据库要手动删除relay log请执行 SET GLOBAL relay_log_purge=0。
